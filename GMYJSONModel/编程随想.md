不过从json的叶子节点类型来看，还是存在对应关系的：
number -> __NSCFNumber
true/false -> __NSCFBoolean
string -> __NSCFConstantString/NSTaggedPointerString
array->__NSArray0/__NSArrayI
object->__NSDictionary0/__NSDictionaryI
null->NSNull
